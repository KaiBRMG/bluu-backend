Deep Link Tester

Purpose: Test if the bluu:// protocol is registered correctly.

Click the button below. If it's working, the Electron app should receive the deep link and log it to the console.

Instructions:

  1. Make sure the Electron app is running
  2. Open this file in your browser: http://localhost:3000/test-deeplink.html
  3. Click the test button above
  4. Check the Electron app console (should show logs from handleDeepLink)
  5. Check if the app receives the OAuth callback

What to look for:

  • Electron console should show: "Deep link received: bluu://callback?code=test123"
  • Should also show: "Parsed URL - pathname: /callback"
  • Should show: "Sending oauth-callback to renderer with code"
